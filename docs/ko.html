본 toolkit이 차단하는 Excel 실수 6가지

지난 3년간 한국·일본·미국의 40여 개 ISO/IEC 17025 calibration lab을 감사하면서 매번 동일한 6가지 실수가 반복되는 것을 확인했습니다. ISO 10012:2026 개정판은 2003년 판이 선택적이었던 항목들을 모두 필수로 강화했기 때문에 이 실수들이 이제 더 audit-relevant 합니다.

  • Bessel skip — 표본 std (STDEV, n-1) 대신 모집단 std (STDEVP, n) 사용
  • k-확장 분모 처리 누락 — 인증서 "U = 0.5 nm at k = 2"를 그대로 표준불확도로 입력 (정답: 0.25)
  • default k = 2 — Welch-Satterthwaite + Student-t 적용 생략
  • 상관 component를 독립으로 취급 — RSS 공식만 사용
  • 비선형 모델에서 감도계수 = 1 default
  • k, νeff 없이 U만 보고 — ISO 10012:2026 6.2가 명시적으로 금지

실제 출력 예시 (CD-SEM 45 nm 선폭)

CDSEM_45nm 예제 시트는 반도체 45 nm 임계치수 측정의 완전한 budget을 생성합니다. A형 반복측정성 + 배율 교정 + 선엣지 거칠기를 합성합니다.

평균 (nm): 45.1125 표준편차 (Bessel): 0.0245 n (반복): 20 u_A (A형): 0.0055 nm (= s / √n, ν = 19) u_B 배율: 0.2500 nm (= 0.5 / 2, ν = 50) u_B LER: 0.2000 nm (= 0.4 / 2, ν = 50) ───────────────────────────────────── u_c (합성): 0.3202 nm ν_eff (Welch-Satterthwaite): 95.5 k @ 95% (Student-t): 1.985 U (확장): 0.636 nm 보고 결과 (ISO 10012:2026 6.2 형식): CD = 45.11 ± 0.64 nm @ 95% (k = 1.99, ν_eff = 95)

5분 워크플로우

  1. GUM_Toolkit_KR_v1.1.xlsx 열기. worked example 시트 또는 빈양식 선택.
  2. B3-B5 셀에 측정량 이름, 단위, 신뢰수준 입력.
  3. 노란색 입력 셀에 각 component 입력 (기호, 값, 분모, 자유도).
  4. 초록색 셀의 uc, νeff, k, U 결과 확인. B17 셀이 ISO 10012:2026 보고 형식 자동 생성.
  5. 24항목 KOLAS 체크리스트로 자체 검증. 완료.
